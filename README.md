# Rico the Humble Gumball

Welcome to **Rico the Humble Gumball**, a lighthearted, text-based adventure game about life inside a classic gumball machine. 

You play as Rico, a simple, unassuming gumball who doesn't want to be the flashiest or the sweetest candy in the machine. While the flashy jawbreakers and sour candies fight for the top spot near the dispenser, Rico just wants to find a comfortable corner, make some friends, and avoid getting chewed.

## Features

* **Choose Your Path:** Navigate the internal politics of the gumball machine. Will you ally with the grumpy Jawbreakers, the energetic Runts, or stay in your quiet corner?
* **Color Customization:** Choose Rico’s color. Does being blue make him calmer? Does being red make him roll faster? (Spoiler: No, he’s just humble).
* **The "Drop Zone" Challenge:** Survive the ultimate test when a quarter drops and the giant metal wheel begins to turn.

## Installation

To play the game locally, clone this repository and run the main script.

```bash
# Clone the repository
git clone [https://github.com/yourusername/rico-the-humble-gumball.git](https://github.com/yourusername/rico-the-humble-gumball.git)

# Navigate to the directory
cd rico-the-humble-gumball

# Run the game
python main.py
